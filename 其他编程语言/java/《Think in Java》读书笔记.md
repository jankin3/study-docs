# 《Think in Java》读书笔记

### 一. 对象导论

讲了一个问题，为什么会有面向对象的设计思想？

1. `解决了什么问题？` 编程的复杂性，有了面向对象的设计，可以简化问题的复杂性

   > 程序员必须建立起在机器模型(位于解空间，就是对问题建模的地方)和实际待解问题模型(位于问题空间，比如业务)之间的关联。建立这种映射是费力的。

2. 如何解决问题？ 

   OOP允许根据问题描述问题，而不是根据计算机描述问题
